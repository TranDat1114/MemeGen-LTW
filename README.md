# Meme Generator NFT

## Introduction

🚀🎉 Welcome to Meme Generator NFT - where your memes become legendary and make you rich (or at least internet-famous)! 😂💎 Have you ever created a meme so good that you thought, "This deserves to be in a museum"? 🖼️ Well, now it can be—on the blockchain! ⛓️ Turn your spicy 🌶️, hilarious 🤣, and chaotic 🤯 memes into NFTs and let the world bid for your humor. One does not simply make memes for free anymore. 💰🔥💸

Meme Generator NFT is a platform that allows users to create, edit memes in the style of Imgflip, and convert them into NFTs. Users can browse, buy, sell, and trade meme NFTs on the platform.

## Features 🎭🔥

- 🎨 **Create Meme** - Unleash your inner meme lord! 🏆: Users can create memes from available templates or upload images.
- 🛠 **Edit Meme** - Make it spicy with crazy fonts and colors! 🌶️: Powerful editing tools with options to change text, font, color, etc.
- 💎 **Convert Meme to NFT** - Immortalize your humor on the blockchain! ⛓️: Each created meme can be minted as an NFT on the blockchain.
- 🛒 **Marketplace** - Buy, sell, and trade memes like a crypto mogul! 🚀: Displays meme NFTs for buying, selling, and trading between users.
- 📂 **Store and View Meme** - Keep your meme treasures safe and sound! 🏦: Supports a personal meme library for reviewing or sharing.
- 🔗 **Crypto Wallet Integration** - Connect your Metamask and make it rain NFTs! 💸: Supports connection with wallets like Metamask for meme NFT transactions.

## Technologies Used (Temp Stack) 🛠️🔥

- **Frontend**: React.js, Next.js, Tailwind CSS
- **Backend**: Next.js

## Installation and Running the Project ⚙️🚀

[My repository](https://github.com/TranDat1114/MemeGen-LTW)

### 1. Clone Repository
```sh
  git clone https://github.com/TranDat1114/MemeGen-LTW
  cd MEMEGEN-LTW
```

### 2. Install Dependencies
```sh
  pnpm i
```

### 3. Run Local Server
```sh
  pnpm dev
```
